Join us for an intensive workshop on using **Git & GitHub**! Whether you are a beginner or looking to re-familiarize yourself with version control, this session will cover everything you need. 

We will talk about:
- Initializing repositories
- Committing and pushing code
- Branching and merging
- How to collaborate using Pull Requests
- Resolving merge conflicts

Please bring your own laptops and have Git installed beforehand. If you need help installing, come a few minutes early so the officers can assist you.
